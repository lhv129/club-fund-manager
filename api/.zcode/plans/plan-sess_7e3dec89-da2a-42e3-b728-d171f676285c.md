### Step 6: Auth Route Handlers

**`src/app/api/auth/login/route.ts`** (POST):
1. Receive `{ login, password }` from body
2. Call Laravel `POST /api/v1/auth/login`
3. On success: `setAuthCookies(response, access_token, refresh_token)`
4. Return `{ user }` (khГөng trбғӘ token vб»Ѓ client JS)

**`src/app/api/auth/refresh/route.ts`** (POST):
1. Read `refresh_token` cookie
2. Call Laravel `POST /api/v1/auth/refresh` with `{ refresh_token }`
3. Update both cookies (rotation)
4. Return success

**`src/app/api/auth/logout/route.ts`** (POST):
1. `clearAuthCookies(response)`
2. Return success
